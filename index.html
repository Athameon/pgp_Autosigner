GPG-Signierung

Herzlich Wilkommen auf der Seite zum automatischen Signieren von deinem public Key.

Bitte gebe nachfolgend deine Daten ein.

Anschließend werden deine Daten überprüft und dein Key wird auf einen Keyserver hochgeladen.

ID des public Keys* z.B.: 0x6B7A6BBE**
Fingerabdruck (Fingerprint)* z.B.: 715A 07FA 1DF1 DC11 E1F2 30A3 1F3F BCD0 6B7A 6BBE**
* Ich bestätige hiermit, dass es sich um meine ID handelt und, dass die Daten richtig und komplett sind.
       *wird benötigt; **Das ist meine eigene ID, die könnt ihr auch gerne unterschreiben

Hinweis:

  - Dein Name (Ein Vorname + Nachname) vom pgp-Schlüssel muss mit deinem Hochschulaccountname übereinstimmen.

  - Dein Schlüssel muss bereits seit einigen Stunden auf einem Keyserver veröffentlicht sein.

  - Das Signieren ist mit deinem Anmeldenamen nur einmal moeglich.

  - Aus Sicherheitsgruenden wird die IP-Adresse, der Loginname, die ID, der Fingerabdruck, die Zeit sowie Browserbasierende Metadaten gespeichert.
